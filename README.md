# Novo Portfolio

A modern developer portfolio built with **TanStack Start**, showcasing projects, skills, experience, and contact information.
The goal of this project is to present my work, experiments, and technical capabilities in a clean and performant web application.

---

## 🚀 Live Demo

🔗 https://novodip.netlify.app/

---

## 🛠 Tech Stack

* **TanStack Start** – Full-stack React framework
* **React** – UI library
* **TypeScript** – Static typing
* **Tailwind CSS** – Styling
* **Framer Motion** – Animations
* **Supabase** – Analytics & small backend utilities
* **Lucide Icons** – Icon set
* **Vite** – Development tooling

---

## ✨ Features

* Responsive modern UI
* Animated sections using Framer Motion
* Resume modal with zoom and download
* Project showcase
* Contact section
* Mobile navigation menu
* Resume click tracking via Supabase
* Smooth scrolling navigation
* Scroll progress indicator

## 🧑‍💻 Running Locally

Clone the repository:

```
git clone https://github.com/yourusername/portfolio.git
```

Install dependencies:

```
npm install
```

Start development server:

```
npm run dev
```

Build for production:

```
npm run build
```

---

## 📊 Analytics

The portfolio tracks resume interactions using **Supabase** to count how many times the resume is opened or downloaded.
